# KeyPROS MVP

공동 임대사업자의 수입, 비용, 정산, 의결 관리를 위한 웹 SaaS MVP입니다.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Vercel
- GitHub
- Node.js LTS

## Branch Rule

- main: 배포용
- dev: 개발 통합용
- feature/*: 기능 개발용

## Commit Convention

- feat: 기능 추가
- fix: 버그 수정
- docs: 문서 수정
- style: 스타일 수정
- refactor: 코드 구조 개선
- chore: 설정 변경

## Getting Started

```bash
npm install
npm run dev
